\doxysubsection{Detailed Description}
A class for running Rektnet inference with Tensor\+RT. 

This class extends the \doxylink{classTensorEngine}{Tensor\+Engine} class to provide functionality specific to \doxylink{classRektNet}{Rekt\+Net}, which is used for detecting keypoints on objects. It handles preprocessing of input data, running inference, and post-\/processing to extract keypoints from the results.
