\doxysubsection{Detailed Description}
A class for running YOLO object detection with Tensor\+RT. 

This class extends the \doxylink{classTensorEngine}{Tensor\+Engine} class to provide functionality specific to YOLO object detection. It handles preprocessing of input frames, running inference, and post-\/processing of results to extract detected objects, such as traffic cones.
